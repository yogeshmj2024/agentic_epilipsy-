Epilepsy Public Health Informatics Framework

Welcome to the Epilepsy Public Health Informatics Framework REST API. This framework provides comprehensive tools for managing patient treatments and analyzing epilepsy disorders in public health settings.

Available Endpoints

POST
/create_patient
Create a new patient record
Example request: { "first_name": "John", "last_name": "Doe", "date_of_birth": "1990-05-15", "gender": "Male" }
POST
/add_epilepsy_diagnosis
Add epilepsy diagnosis to a patient
Example request: { "patient_id": "patient-uuid", "epilepsy_type": "Focal", "severity_level": "Moderate" }
GET
/list_patients
List all patients
GET
/patient_summary/<patient_id>
Get comprehensive summary for a patient
POST
/add_seizure_event
Add a seizure event to a patient
Example request: { "patient_id": "patient-uuid", "seizure_date": "2024-06-01T10:30:00", "seizure_type": "Focal Aware", "duration_minutes": 2.5, "severity": 3 }
GET
/statistics
Get overall framework statistics

Use tools like curl, Postman, or any HTTP client to interact with these endpoints.
